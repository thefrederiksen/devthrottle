Issue #293 — Copy Handover Info & Help dialog advertise the tailnet front door (not loopback)

Developer Agent proof · branch issue-293-advertise-tailnet-control-endpoint · verified live on SOREN_NORTH (Tailscale up, MagicDNS soren-north.taildb08ed.ts.net).

What was implemented

Two UI surfaces that produce text handed to a remote consumer hardcoded http://127.0.0.1:<port> for the Control API address — useless from any other machine. They now advertise the Tailscale Serve front door https://<magicdns>:<port> when this node is on a tailnet, and only a clearly-labelled loopback value when Tailscale is genuinely unavailable.

  • src/CcDirector.Core/Network/TailscaleIdentity.cs — added ResolveAdvertisedControlApiEndpoint(int port) (the single policy) and the pure, unit-tested FormatAdvertisedControlApiEndpoint(string?, int) combiner.
  • src/CcDirector.Avalonia/MainWindow.axaml.cs — CopySessionNameAndId ("Copy Handover Info") now resolves the endpoint off the UI thread and emits it.
  • src/CcDirector.Avalonia/HelpDialog.axaml.cs — the About dialog shows the dialog immediately with a (resolving...) placeholder and fills the "Control endpoint" row + Copy text in an Opened handler (Responsive-UI).
  • src/CcDirector.Core.Tests/TailscaleIdentityTests.cs — 4 new test cases.

Design decision (deviation from the issue's literal plan, with rationale)

The issue proposed adding TailscaleIdentity.TryGetControlApiEndpointForPort(port). That helper already effectively exists: TryGetFrontDoorUrlForPort(int) already returns exactly https://<magicdns>:<port> or null, backed by the pure, already-unit-tested BuildFrontDoorUrlForPort. Adding the proposed method would duplicate the front-door policy — the opposite of the issue's own stated goal ("the policy lives in ONE place"). So this change reuses TryGetFrontDoorUrlForPort and centralises only the advertised-endpoint formatting (front-door-or-labelled-loopback) in one place, which the literal plan would have duplicated across both call sites. Every behavioral acceptance criterion is met.
