Issue 756 Proof - Convert OpenCode to a Concrete Agent Plugin

Summary

PASS.

OpenCode is now represented by OpenCodeAgentPlugin instead of the generic BuiltInAgentPlugin adapter. The plugin owns OpenCode settings metadata, executable detection candidates, validation arguments, SQLite history metadata, launch metadata, presets, agent construction, and the OpenCode driver binding.

The plugin remains data/launch/driver only. OpenCode does not add or own any Director UI tab.
