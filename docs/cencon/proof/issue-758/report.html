Issue 758 Proof - External Agent Plugin Loading

Summary

PASS.

Added manifest-based external plugin loading for the current agent plugin contract. A plugin folder may contain plugin.json files that point at a DLL and concrete IAgentPlugin type. The loader validates contract version, manifest identity, plugin construction, duplicate ids, duplicate agent kinds, RawCli reservation, and driver kind matching.

The current IAgentPlugin contract still uses the fixed AgentKind enum, so external plugins are intentionally constrained to known agent kinds until a future contract version adds dynamic agent type ids.
