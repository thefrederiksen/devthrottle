Issue 757 Proof - Convert Grok to a Concrete Agent Plugin

Summary

PASS.

Grok is now represented by GrokAgentPlugin instead of the generic BuiltInAgentPlugin adapter. The plugin owns Grok settings metadata, executable detection candidates, validation arguments, transcript-file history metadata, launch metadata, presets, agent construction, and the Grok driver binding.

The plugin remains data/launch/driver only. Grok does not add or own any Director UI tab.
