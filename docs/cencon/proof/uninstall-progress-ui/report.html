Nicer uninstall progress UI

Setup wizard (tools/cc-director-setup) - replaces the raw MessageBox pop-ups + frozen window with an in-wizard confirm → live progress → completion flow.

Before

  • A grey Windows MessageBox "Uninstall CC Director? ... Continue?" (the ugly pop-up).
  • The window then sat frozen-looking with NO progress while the app, tools, PATH, skills, tasks, autostart, and Tailscale mapping were removed.
  • A second grey MessageBox for the result.

After (this change)

  • Engine: Uninstaller.Apply(role, IProgress<string>? progress) reports a friendly message as each phase begins (back-compat - the param is optional).
  • UI: a themed UninstallStep with three views - confirm, live progress, completion - matching the install flow. No MessageBox.

Fig 1 - Confirm (themed, not a pop-up)

"What gets removed" list + a "your data is kept" note; Cancel + a red Uninstall button.

Fig 2 - Live progress

Current action ("Removing the Start Menu shortcut...") + an indeterminate bar + completed steps accumulating with green dots.

Fig 3 - Completion

Green dot + "Uninstall complete", a one-line summary, the data-kept note, and a Close button. (On a partial failure it instead shows the specific items that could not be removed.)

How it was captured

The three views were screenshotted live by driving the wizard with a FAKE runner (no real removal) via a temporary, env-var-gated demo trigger that was removed before commit. The real destructive end-to-end is verified on a disposable machine (same as the rest of #257).
