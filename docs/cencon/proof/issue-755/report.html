Issue 755 Proof - Convert Gemini to a Concrete Agent Plugin

Summary

PASS.

Gemini is now represented by GeminiAgentPlugin instead of the generic BuiltInAgentPlugin adapter. The plugin owns Gemini settings metadata, executable detection candidates, validation arguments, terminal-buffer history metadata, launch metadata, presets, agent construction, and the Gemini driver binding.

Gemini history remains data-only: historyProvider=TerminalBuffer. The visible History tab is still Director-owned UI, verified by the existing plugin assembly guardrail test and live screenshot.
