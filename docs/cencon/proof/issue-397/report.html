What was implemented

The src/CcDirector.Avalonia.Tests project had an active .csproj and current test files, but was not referenced by cc-director.sln. Its tests therefore never ran in a normal solution build or test. The project was added to the solution using dotnet sln cc-director.sln add ... --solution-folder src, which inserted the project entry, all six build configurations (Debug/Release x Any CPU/x64/x86), and the src solution-folder nesting that matches every other project in the solution. The single changed file is cc-director.sln (15 lines added, no other files touched).
