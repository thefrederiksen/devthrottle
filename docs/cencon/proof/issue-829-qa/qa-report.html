Method (independent reproduction)

Drift was created against the real install by injecting a known orphaned legacy alias shim (%LOCALAPPDATA%\cc-director\bin\cc-send.cmd, one of PythonToolsInstaller.LegacyAliasShimNames) and toggling tools.autoUpdate.enabled. To hold the orange state stable for capture and to drive the repeated-failure path, the injected shim was marked read-only so the reconcile's best-effort purge could not remove it; clearing the read-only attribute then let the next backoff retry self-heal to green. The user's config was backed up and restored afterward; all injected shims were removed.
