Issue #721 - cc-spawn - Developer Agent Proof

Branch: issue-721-cc-spawn. A command-line way to open a session on the local Director and then message/ask it.

What was implemented

  • tools/cc-spawn/ - new tool: cc-spawn <repo> [--agent --prompt --name --type --command --command-args] POSTs NewSessionRequest to $CC_DIRECTOR_API/sessions, prints the new session's short id + full GUID. --name applies a follow-up rename (PATCH).
  • cc_shared/director.py - added patch_json (for the rename).
  • Shipped + self-verified: added to tools/registry.json (ship:true), src/CcDirector.Core/Tools/tools-manifest.json, docs/cli-reference.md, and the FleetToolsShipGuardTests guard array.
