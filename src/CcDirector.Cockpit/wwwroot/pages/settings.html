Startup

Registers a Run entry under HKCU so the Gateway starts in your session when you log in. The fleet only works while you are logged in, so this is per-user, never a machine service.

Brain warm claude.exe

A hosted claude.exe used by Gateway agents (the wingman). It starts on first use; restart replaces a wedged or dead brain. The model is pinned via config.json brain_model; a change applies on the next Gateway restart.
